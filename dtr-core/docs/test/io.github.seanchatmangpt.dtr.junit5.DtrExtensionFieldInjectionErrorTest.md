## Field Type Validation

Field injection validates that annotated fields have type DtrContext.

> [!NOTE]
> If field type is incorrect, a clear IllegalStateException is thrown.

> [!WARNING]
> Ensure @DtrContextField is only used on fields of type DtrContext.

| Key | Value |
| --- | --- |
| `Valid Field Type` | `DtrContext` |
| `Error` | `IllegalStateException with descriptive message` |
| `Invalid Field Type` | `Any other type (String, Object, etc.)` |

## Unannotated Fields

Fields without @DtrContextField are not injected.

> [!NOTE]
> Only fields explicitly marked with @DtrContextField receive injection.

| Key | Value |
