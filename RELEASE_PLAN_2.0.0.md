### Update Strategy

**Single-Point Update**: Update the version in the root `pom.xml` only. Child modules inherit via `<parent>` reference and will automatically use the parent version.

```xml
<!-- Root pom.xml (line 14) -->
<version>1.1.12-SNAPSHOT</version>
↓
<version>2.0.0</version>
```

Child modules reference parent version via:
```xml
<parent>
    <groupId>org.r10r</groupId>
    <artifactId>doctester</artifactId>
    <version>1.1.12-SNAPSHOT</version>  <!-- Automatic update via parent -->
</parent>
```

## 3. Breaking Changes Summary

### For Library Users

| Breaking Change | Impact | Migration |
|-----------------|--------|-----------|
| **Markdown Output** | HTML docs no longer generated by default | Use Markdown or add custom HTML renderer plugin |
| **JUnit 5 Required** | Must use `@ExtendWith(DocTesterExtension.class)` | See migration guide in CHANGELOG_2.0.0.md |
| **HttpClient 5.x** | Internal HTTP APIs changed | Update any custom TestBrowser implementations |
| **Removed HTML Classes** | `RenderMachineImpl` deleted | Implement custom `RenderMachine` interface |

### For CustomTestBrowser Implementers

- Update imports: `org.apache.httpcomponents.client5` (was `org.apache.httpcomponents`)
- Update URI handling to use HttpClient 5.x API
- Request building patterns remain similar (fluent API preserved)

## 4. New Features in 2.0.0

### Major Features
- **Annotation-Based API**: `@DocSection`, `@DocDescription`, `@DocNote`, `@DocWarning`, `@DocCode`
- **Markdown-First Output**: Better version control and documentation generation
- **JUnit 5 Integration**: Full Jupiter support with property-based testing (jqwik)
- **WebSocket/SSE Support**: New test clients for real-time protocols
- **Advanced Authentication**: Bearer tokens, API keys, Basic Auth providers
- **OpenAPI Generation**: Automatic schema extraction from tests
- **Java 25 Modernization**: Records, sealed classes, text blocks, virtual threads

### Dependency Updates
- Apache HttpClient: 4.5 → 5.6
- Jackson: 2.5.4 → 2.21.1
- JUnit: 4.12 → 5/Jupiter 6.0.3
- Mockito: 4.11.0 → 5.22.0
- SLF4J: 1.7.12 → 2.0.17
- Ninja Framework: 6.x → 7.0.0
