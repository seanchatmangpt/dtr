# DTR Observatory Facts

Generated by `make observe`. Refreshed automatically on session start (if binary is built).

**Why**: Each fact file is ~50 tokens. Running grep sweeps costs ~5000 tokens. Use facts first.

## Files

| File | Contents | Use When You Need To Know... |
|------|----------|------------------------------|
| `modules.json` | Maven version, modules, Java release | Project version, which modules exist |
| `java-profile.json` | Compiler settings from .mvn/maven.config | Java release, preview flags |
| `rust-capabilities.json` | Rust crates, binaries, test counts | Which Rust binaries are available |
| `source-stats.json` | Java file counts by module | How many Java files per module |
| `tests.json` | Test class counts by module | How many test classes exist |
| `guard-status.json` | H-Guard scan result (GREEN/RED/UNKNOWN) | Are there H-Guard violations right now? |

## Commands

```bash
make build-observatory   # Compile dtr-observe binary (once, after first clone)
make observe             # Refresh all fact files
make observe-test        # Run Observatory Rust unit tests
```

## Fact Formats

**modules.json**
```json
{"version":"2026.1.1-rc.1","groupId":"io.github.seanchatmangpt.dtr","modules":["dtr-core","dtr-benchmarks"],"main_module":"dtr-core","java_release":"26","enable_preview":true}
```

**guard-status.json**
```json
{"status":"GREEN","violations":0,"scanned_files":75,"generated":"2026-03-14T10:00:00Z"}
```
Status values: `GREEN` (clean), `RED` (violations), `UNKNOWN` (binary not built).

**rust-capabilities.json**
```json
{"crates":[{"name":"dtr-guard","bins":[{"name":"dtr-guard-scan","built":true}],"tests":14}],"total_tests":72}
```
