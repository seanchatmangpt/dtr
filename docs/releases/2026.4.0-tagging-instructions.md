   - Publish via Sonatype Central Publisher Portal
   - Wait for publication confirmation

3. **Release Complete**

## Rollback Procedure (If Needed)

If something goes wrong after pushing the tag:

### Delete the local tag
```bash
git tag -d v2026.4.0
```

### Delete the remote tag
```bash
git push origin :refs/tags/v2026.4.0
```

**Note:** If artifacts have already been published to Maven Central, they cannot be removed. You would need to release a new version (e.g., 2026.4.1) to fix any issues.
