## What Happens After Push

1. **GitHub Actions Trigger:** The `v2026.4.0` tag triggers the GitHub Actions workflow
2. **CI Verification:** Pipeline runs `mvnd verify --enable-preview`
3. **Maven Central Publish:** On successful verification, artifacts are published to Maven Central
4. **Release Complete:** Version 2026.4.0 becomes available to users
