# `UsPatentTemplate`

> **Package:** `io.github.seanchatmangpt.dtr.rendermachine.latex`  

USPTO-compatible LaTeX template for patent exhibits and technical briefs. Uses report class with formal numbering, large margins, explicit section numbering, and claim-style formatting suitable for patent offices and enterprise technical documentation. Formatted for single-column layout with emphasis on clear, evidence-based documentation of technical implementations.
