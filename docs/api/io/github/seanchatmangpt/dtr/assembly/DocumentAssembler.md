# `DocumentAssembler`

> **Package:** `io.github.seanchatmangpt.dtr.assembly`  

Assembles multiple DocTest .tex files into a unified document manifest. This class provides static factory methods for analyzing and combining LaTeX source files generated by DTR tests into a consolidated assembly manifest with aggregated metadata.

```java
public final class DocumentAssembler {
    // assemble, countNewPages, countCodeListings, countTables, countCitations, countCrossReferences
}
```

---

## Methods

### `assemble`

Assembles multiple .tex files into a manifest with aggregated metadata.

| Parameter | Description |
| --- | --- |
| `texFiles` | list of paths to LaTeX source files |

> **Returns:** AssemblyManifest containing aggregated metadata

| Exception | Description |
| --- | --- |
| `IllegalArgumentException` | if texFiles is null or empty |

---

### `countCitations`

Counts citation commands (\cite{...}).

### `countCodeListings`

Counts code listings (lstlisting environments).

### `countCrossReferences`

Counts cross-references (\ref{...} and \label{...}).

### `countNewPages`

### `countTables`
