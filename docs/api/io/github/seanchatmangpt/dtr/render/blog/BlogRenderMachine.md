# `BlogRenderMachine`

> **Package:** `io.github.seanchatmangpt.dtr.render.blog`  

Blog post render machine generating platform-specific markdown. Converts test execution into blog post markdown compatible with Dev.to, Medium, Substack, LinkedIn, and Hashnode. Generates social media queue entries for tweets and platform-specific content.
